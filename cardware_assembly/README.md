# Cardware Assembly

### What you need:

- Cardware parts
- 3D-printed parts
- Servo motor (regular 180-degree), along with 21mm length horn
- Servo motor (continuous 360-degree), along with "+" shaped horn
- Zip ties (50)
- Shear cutters or pliers (to trim the zip ties)

- [Drivetrain](#drivetrain)
- [Steering](#steering)
- [Camera](#camera)
